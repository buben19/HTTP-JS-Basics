OOP

Object-oriented programming

Most used paradigm in programming

Application is divided into objects

Object contains data and code (methods)

Construction of object instance using new keyword

Object can access itself using this keyword

JS Object

const person = {
	firstName: "John",
	lastName : "Doe",
	id       : 5566,
	fullName : function() {
		return this.firstName + " " + this.lastName;
	}
};

console.log(person.fullName());

JS Class

class Person {
  constructor(firstName, lastName, id) {
    this.firstName = firstName;
    this.lastName = lastName;
    this.id = id;
  }

  getFullName() {
    return this.firstName + " " + this.lastName;
  }
}

let person = new Person("John", "Doe", 5566);
console.log(person.getFullName());
